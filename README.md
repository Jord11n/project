The only project I can showcase that I'm proud of is a game of Minesweeper; it took me a while to do, mainly because I had to learn how it worked game-wise first. Playing some versions of Minesweeper with friends helped a lot especially when I was taught on how it worked, so I wanted to bring that to life here.

Uses @keyframes to manipulate the transform: translate() property, which helps create a jittery effect for the mines. For the slide-up screen, I used the transitionend event in JavaScript to coordinate the "Slide Up" and the "GO!" flash.
